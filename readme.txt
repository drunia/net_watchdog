NetWatchDog - Утилита помогающая отслеживать состояние связи с хостом/устройством
с помощью звукового оповещения в случае наступления отслеживаемого состояния.

Остлеживание производится по нескольким методам:
1. PING - используется протокол ICMP для проверки доступности хоста/устройства
    Простой пинг устройсва.

2. PORT - используеься возможность соеденится с портом на удаленном устройстве по протоколу TCP.
    В этом случае удобно проверять доступность хоста/устройсва - проверив например HTTP (80/TCP) порт,
    этот метод может понадобиться, когда проверка методом PING невозможна (протокол ICMP заблокирован фаерволом).

3. ONVIF - это расширенный метод (с картинкой) проверки устройства поддерживающего протокол ONVIF.
    Например позволяет видеть картинку с отслеживамой камеры.


Утилита позволяет отслеживать как онлайн соятоние хоста/утсройсва (пока оно находится в сети),
так и оффлайн (пока оно находится не в сети) и при изменении отслеживаемого состояния уведомить Вас звуком.
